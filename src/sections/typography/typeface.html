LifterLMS uses two typefaces, Nexa and Montserrat.

Nexa is reserved exclusively for use in the LifterLMS logo while Montserrat is used for all other text in design for both web and print.

The Nexa family is characterized by excellent legibility in both web & print design areas. It's well-finished geometric design makes it ideal for use in both headlines.

Montserrat is a free and open source font inspired by the urban typograhpy found in posters in the Montserrat neighborhood of Buenos Aires. The font is licensed under, the SIL Open Font License. It is made available via Google Fonts and a GitHub project.
